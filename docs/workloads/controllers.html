Controllers

Controllers provide the logic to manage virtual machine instances in a way that addresses specific use-cases:

  • VirtualMachineInstanceReplicaSet: Replicating stateless Virtual Machines.
  • VirtualMachine: Stateful Virtual Machine, similar to a StatefulSet with replicas set to 1.
